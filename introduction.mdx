## What is Useawards SDK?
Useawards SDK is a React library that provides a set of tools for integrating Useawards services into your application. It consists of three packages: Client, React dom, and Core.

## What is this documentation for?
This documentation is intended to help developers use the Useawards SDK to build applications with Useawards services. It provides an overview of the library's capabilities and detailed instructions for installation, configuration, and usage.

## Who should use this library?
This library is designed for developers who want to integrate Useawards services into their React applications. It is assumed that users have a basic understanding of React and web development.
